## 1. 必备通用能力

### （1）Python 几乎是硬要求

### （2）大模型应用基础必须会

高频出现的关键词：

- Prompt Engineering
- RAG
- Agent
- Fine-tuning
- 工具调用
- 多模态
- 对话系统
- 知识库问答

也就是说，企业普遍希望你不只是“会调 API”，而是知道：

- Prompt 怎么设计和迭代
- RAG 怎么搭、怎么评估召回与答案质量
- Agent 怎么做任务拆解、规划、工具调用、状态管理
- 模型微调适合什么场景
- 怎么在成本、延迟、准确性之间做权衡

---

### （3）工程化能力非常重要

这是很多候选人容易忽视、但 JD 里非常高频的一点。

企业要的不是 demo，而是**可交付系统**，因此会看：

- 服务化封装
- API 集成
- 前后端对接
- Docker 容器化
- Linux 环境
- CI/CD
- 部署与监控
- 性能优化
- 自动化流程

一句话说：

> 你不仅要“把模型调通”，还要“把系统跑稳、跑快、跑上线”。

---

## 2. 高频专项能力

### （1）RAG 能力

这是最普遍的能力要求之一。

通常希望你会：

- 知识库构建
- 文档清洗、切分、标注、结构化

- 向量库选型与使用

    - FAISS
    - Pinecone
- 检索优化
- 重排/召回优化
- 回答准确性与幻觉控制
- 针对真实场景进行效果评估


---

### （2）Agent 能力

Agent 是这些 JD 的第二大核心。

常见要求包括：

- 智能体架构设计
- 任务规划
- 对话管理

- 流程编排
- 工具调用 / API 调用
- 多智能体协作

- 工作流自动化

---

### （3）模型部署与优化能力

# 四、他们需要的项目经历

如果你要准备简历或面试，最值钱的项目经历通常是下面几类。

## 1. RAG / 知识库问答项目

这是最通用、最容易匹配多家 JD 的项目类型。

理想项目特征：

- 你搭过完整 RAG 流程
- 包括数据清洗、切分、向量化、检索、生成、评估
- 用过 Milvus / FAISS / Pinecone 等向量库
- 做过召回率、准确率、延迟优化
- 解决过幻觉、知识过期、长文档、多源文档问题

这类项目可以对应：

- 企业知识库问答
- 文档问答
- 智能客服
- 技术支持助手
- 内部知识助手

---

## 2. Agent 系统项目

这是最贴“AI Agent应用相关岗位”的经历。

高价值项目包括：

- 基于 LangChain / LangGraph / AutoGen / Semantic Kernel / LlamaIndex / Dify 的 Agent
- 有任务规划、工具调用、记忆、状态管理
- 多 Agent 协作
- 实现复杂任务闭环，而不是单轮问答

理想案例：

- 智能客服 Agent
- 技术支持诊断 Agent
- AI Coding Agent
- 自动化办公 Agent
- 流程审批/工单处理 Agent
- 多工具协同 Agent（搜索、数据库、接口、执行器）

---

## 3. 大模型产品从 0 到 1 落地项目

很多 JD 明确写了“0到1”“完整交付”“主导落地”。

这类项目特别加分，因为说明你不只是参与一个点，而是完整负责过：

- 需求分析
- 架构设计
- 模型/技术选型
- 开发实现
- 上线部署
- 效果优化
- 与业务协同推进

这比“我做过一个小 demo”含金量高很多。

---

## 5. AI Coding / 技术支持 / 企业助手类项目

从 JD 看，这是非常热门的一条线。

很受欢迎的项目方向包括：

- 代码补全/代码生成
- 缺陷定位
- 日志分析
- 自动测试用例生成
- API / 文档自动生成
- IDE 插件
- 企业内部研发助手
- CRM / 工单 / 客服诊断系统

这类项目特别容易体现：

- Agent 能力
- 工具调用
- 知识检索
- 业务价值
- 工程复杂度

---

## 6. 多模态 / 语音 / 具身智能相关项目

这不是所有岗位都要求，但属于明显加分项。

例如：

- 文本 + 图片检索/问答
- 语音对话系统
- 多模态理解
- 机器人交互
- 环境理解与语音交互
- 文档多模态检索

如果你投的是偏前沿团队、研究院、算法专家岗，这类项目价值更高。

---

# 六、最终浓缩版结论

这些 AI Agent / 大模型应用岗位，本质上最想招的是这类人：

> **既懂 LLM 应用范式（Prompt、RAG、Agent），又有扎实 Python 和工程化能力，还做过真实业务项目落地的人。**

他们最看重的三件事：

## 1. 背景

- 计算机/AI相关专业，本科到硕士
- 1-5 年开发或算法经验
- 有 LLM / NLP / Agent / RAG 实战经历

## 2. 能力

- Python + 工程能力
- RAG、Agent、Prompt、模型部署与调优
- Linux/Docker/向量数据库/服务开发
- 业务理解、协作与快速迭代能力

## 3. 项目经历

- 知识库问答/RAG 系统
- Agent 智能体系统
- 开源大模型部署优化
- AI 应用从 0 到 1 落地
- 智能客服、AI Coding、企业助手、多模态应用
