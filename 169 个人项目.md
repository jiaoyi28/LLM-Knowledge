# 2 小艺个人知识图谱

个人知识图谱：
1. 采用两层结构：原子事件和聚类事件，比如一次去上海的出差是一个聚类事件，过程中的预定上海机票、预定上海酒店、上海打车、返程分别为一个个原子事件；原子事件的粒度主要以产品的需求来确定，通过prompt来调整
2. 事件采用三层分类：
	1. 生活：
		1. 出行：公共交通、自驾、
		2. 购物：线上、线下
		3. 餐饮：聚餐、外卖
		4. 社交
		5. 娱乐
		6. 医疗
	 2. 工作：
		 1. 差旅
		 2. 工作内容
		 3. 职业发展
	 3. 教育：
		 1. 学习
		 2. 考试
		 3. 学业成就
3. schema设计
	1. 四种节点：人物、事件、地点、事件类型
4. 构建方案
	1. 数据聚合：数据预处理，将多数据源进行过滤和聚合（地点、时间、语义），将相关的事件数据放在同一分组，增强LLM抽取的效果，控制上下文大小。单一的按数据源、时间分组效果均不好
	2. 原子事件抽取：给定schema，让LLM进行原子事件的批量抽取，需要控制单个batch的数据条数，避免超出上下文窗口
	3. 事件聚类与摘要生成：将抽取的原子事件通过LLM进行聚合
